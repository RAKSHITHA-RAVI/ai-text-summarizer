# 📝 AI Text Summarizer

A web app that generates concise summaries of long text using **FastAPI (Python)** for the backend and **React (Material-UI)** for the frontend.  
It uses **Hugging Face Transformers (BART model)** for summarization.

---

## 🚀 Features
- Frontend built with **React + Material-UI**
- Backend with **FastAPI + Hugging Face**
- **Dark/Light mode toggle**
- **Word count** tracking
- **Custom summary styles**: Short, Medium, Detailed
- **Copy / Download summary**
- **History of previous summaries**
- Responsive and modern UI
